# 事件池（中文版对照）

这是一个持续维护的事件池，用来防止后续判断遗漏宏观、地缘、交易所、代币解锁、产品规则变化等关键事件。

## 核心原则：事件池不能无限膨胀

事件池不是聊天记录，也不是新闻堆积箱。它的作用是让每次决策先知道“未来几天到几周哪些事情可能改变价格路径”。为了不影响上下文判断，维护时按下面规则处理：

- **每次交易决策只优先读取活跃区**：先看“当前高优先级事件监控”和“未来 7-30 天事件”，历史区只在复盘或寻找历史相似案例时读取。
- **重复事件写成规则，不每个月机械复制**：CPI、PPI、PCE、非农、FOMC、初请失业金、期权交割、国债拍卖、代币解锁这类固定事件，应放进“重复事件规则”，只有临近交易窗口时再展开成具体日期。
- **已过期事件要移动**：事件过期后不能继续留在 upcoming，要改成 past / resolved，并只保留对交易有用的结论，例如“是否引发波动、市场如何定价、以后要吸取什么教训”。
- **突发事件单独置顶**：战争、停火、制裁、油价冲击、稳定币脱锚、交易所宕机、黑客攻击、监管执法等，优先级高于普通宏观日历。
- **每次实时判断必须重新检索**：事件池只是记忆，不是事实源。涉及最新价格、最新宏观、最新新闻、最新交易所数据时，必须重新 web search 或调用交易所/API 数据。

## 优先级分层

| 优先级 | 类型 | 进入决策的方式 |
|---|---|---|
| P0 | 突发重大事件：战争升级、油运受阻、稳定币脱锚、交易所风险、监管禁令、系统性金融风险 | 直接改变主判断，必要时立即重算多空胜率 |
| P1 | 未来 72 小时内的 FOMC、CPI、PPI、PCE、非农、鲍威尔讲话、重大期权交割 | 必须进入当前交易计划，明确是否能持仓穿越事件 |

| P3 | 长期叙事：降息周期、美元流动性、财政赤字、机构配置、链上生态增长 | 作为背景，不覆盖短线价格结构和衍生品拥挤度 |

## 每月/季度重复事件规则

这些事件不是每次都重新写一遍，而是在靠近交易窗口时展开成具体事件：

- **CPI**：通常每月公布一次，市场重点看核心 CPI、超级核心服务、住房通胀、能源项。影响链条是通胀数据 -> 降息/加息预期 -> 美债收益率和实际利率 -> 美元和风险资产估值 -> BTC/山寨币。
- **PPI**：通常每月公布一次，偏上游价格。若 PPI 明显升温，市场会担心后续 PCE 或企业成本压力。
- **PCE**：美联储偏好的通胀指标，尤其核心 PCE。对中期利率路径影响较大。
- **非农 NFP / 失业率 / 工资增速**：决定市场是交易“经济过热、降息推迟”，还是“经济走弱、衰退风险上升”。两种坏数据可能方向不同：温和走弱有利降息，失控走弱反而触发避险。
- **FOMC / 点阵图 / 鲍威尔发布会 / 会议纪要**：不是只看是否降息，更要看点阵图、通胀措辞、就业风险权重和金融条件态度。
- **初请失业金**：频率更高，常用于观察就业市场是否突然恶化。
- **美国国债拍卖和季度再融资**：影响长端收益率和流动性，若拍卖疲软可能压制风险资产。
- **BTC/ETH 期权交割**：月度/季度交割容易影响短线波动，尤其当最大痛点、Gamma 集中区和现货价格接近时。
- **代币解锁**：不是每个解锁都必跌，要看解锁规模、流通市值占比、接收方、是否转入交易所、当时市场流动性和相对强弱。
- **ETF 资金流**：不能只看单只基金，要看全市场净流入/流出，并结合 BTC 价格是否吸收卖压。

## 事件类别

### 宏观定期事件

- FOMC、会议纪要、SEP / 点阵图。
- CPI、PPI、PCE。
- 非农、初请失业金、失业率、工资增速。
- 美国国债再融资、长债拍卖、收益率曲线变化。
- BOJ、ECB、PBOC 等主要央行决议。

### 宏观突发事件

- 战争、停火、制裁、能源运输受阻。
- 油价供给冲击。
- 银行、信用、商业地产或影子银行风险。
- 债务上限、财政赤字、主权信用评级变化。
- 央行突然干预汇率或流动性。

### 加密定期事件

- BTC/ETH 期权交割。
- ETF 净流入/净流出。
- 代币解锁。
- 公链升级、硬分叉、重要生态激励。
- 交易所上币、下架、规则调整。

### 加密突发事件

- 黑客攻击、跨链桥漏洞、协议坏账。
- 稳定币脱锚。
- 交易所宕机、提款异常、资不抵债传闻。
- 监管执法、诉讼、禁令。
- 巨鲸、基金会、项目方大额转入交易所。

### 产品特定事件

- **BTC**：ETF 净流、市场主导率、矿工/上市公司/国库持仓变动、期权交割、美元流动性。
- **ETH**：ETH/BTC 趋势、ETF/质押/监管变化、L2 和链上活跃度。
- **SOL**：SOL/BTC 和 SOL/ETH 相对强弱、生态资金流、链稳定性、重大解锁或宕机风险。
- **非核心资产和特殊产品**：只有用户明确点名时才追踪，避免默认决策被噪音干扰。

## 更新规则

新增事件时必须写清楚：

- 绝对日期和时区。
- 来源和置信度。
- 对多头和空头分别意味着什么。
- 什么现象说明这个事件真的在影响市场。
- 什么现象说明这个事件已经被市场消化或判断失效。
- 下一次需要更新的时间。

## 历史事件备注

### 2026-06-15 美国/伊朗初步协议、霍尔木兹重开观察

- AP 报道美国/伊朗达成初步协议，内容包括结束战争、延长脆弱停火、开放霍尔木兹海峡；但执行依赖后续正式签署。
- 多头路径：正式签署并执行 -> 油价风险溢价下降 -> 通胀担忧缓和 -> 降息预期改善 -> BTC/ETH/SOL 风险偏好修复。
- 空头路径：签署推迟、再次交火、航运保险/通行仍受阻、核谈判或制裁分歧扩大 -> 油价反弹 -> 通胀预期和避险情绪回升。
- 复盘要点：地缘新闻不能把“标题”当“执行完成”，必须继续看油价、美债收益率、DXY、VIX 和官方跟进。

### 2026-06-12 SPCX / SpaceX Listing

- SpaceX / SPCX 上市和 OKX SPCX-USDT 转换造成事件波动。
- 复盘要点：必须区分真实股票、OKX 永续合约、标记价、指数价和最新成交价，不能把它们当成同一个东西。

### 2026-06-10/11 CPI/PPI Shock

- CPI/PPI 显示能源和上游通胀压力。
- 复盘要点：不能只看标题数字，要继续拆核心项、能源项、油价走势和美联储预期变化。
